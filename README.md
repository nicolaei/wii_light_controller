# Light Game
This is a python thingy created to show at expos for the University of Oslo.

## Dependencies
### Hardware
Open Lighting Architecture supported DMX device
Wii-mote
Bluetooth dongle

### Python dependencies
Python 2.7
cwiid
ola (open lighting architecture)

For cwiid:
`apt-get install libcwiid1 lswm wmgui wminput`
